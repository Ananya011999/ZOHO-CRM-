# ZOHO-CRM-
Manual testing project based on Zoho CRM functionalities, covering modules like campaign,leads,vendors,contacts,product,potentials,accounts,activity and reports. Includes test cases, scenarios, and bug reports. Performed exploratory,functional,integration,system,regression, and UI testing to validate workflows and ensure application quality with proper defect tracking.
User TestRail to write the scenarios and JIRA for defect tracking.
